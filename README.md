# ooxx 

Welcome to ooxx, a futuristic space-themed version of the classic Tic Tac Toe game! Play this game in a visually stunning environment inspired by the sci-fi aesthetics of the Halo universe.

## Play Online

Play Tic Tac Toe - OOXX Now! [here](https://ynot1996.github.io/ooxx/).

## Technologies Used

- HTML5: For structuring the game layout.
- CSS3: For styling and creating the futuristic space theme, including animations and responsive design.
- JavaScript: For implementing the game logic, including turn-based mechanics, win conditions, and user interactions.
